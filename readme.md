CatMash is a under-development facemash-like php project, to vote for the cutest cat ever! 
### Requirements

- PHP >= 7.1
- Composer

### Setup

```bash
git clone git@github.com:mathieutu/catmash.git
composer install
# Create a .env file with database configuration
php artisan migrate
```
